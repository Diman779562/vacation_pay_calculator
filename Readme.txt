Для задания параметров в приложении требуется задать GET запрос в формате
 
http://localhost:8080/calculacte?vacation_days=10 &average_salary=1000 &holiday_start_date=17.10.2022
 
где: vacation_days - количество отпускных дней
     average_salary - средняя зарплата за год
     holiday_start_date - день начала отпуска в формать д.м.гггг
